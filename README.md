# 📚 Os Sete Maridos de Evelyn Hugo

Uma página web desenvolvida em **HTML** e **CSS** sobre o livro **Os Sete Maridos de Evelyn Hugo**, de Taylor Jenkins Reid. O projeto apresenta informações sobre a obra de forma organizada e com um design agradável.

## 📁 Estrutura do Projeto

```text
📦 projeto
├── index.html
├── pages
│   ├── curiosidades.html
│   ├── historia.html
│   └── personagens.html
├── style
│   └── style.css
└── README.md
```

## 📄 Páginas

### 🏠 `index.html`
Página inicial do projeto. Apresenta uma introdução ao livro e serve como ponto de acesso para as demais páginas.

### 📖 `pages/historia.html`
Contém um resumo da história do livro, apresentando a trama principal sem comprometer a navegação do usuário.

### 👥 `pages/personagens.html`
Apresenta os principais personagens da obra e suas respectivas importâncias na narrativa.

### ✨ `pages/curiosidades.html`
Reúne curiosidades e informações interessantes sobre o livro e sua autora.

## 🎨 Estilo

O arquivo `style/style.css` é responsável pela estilização de todas as páginas do projeto, garantindo uma identidade visual consistente.

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
